# AZT-NO – Setup and Execution Guide

This document provides step-by-step instructions to set up and run the AZT-NO (Adaptive Zero-Trust Network Observer) project on a local system. The setup is designed to be reproducible and platform-independent.

---

## 1. System Requirements

- **Operating System:**  
  Windows 10 / Windows 11 (Linux Ubuntu recommended for advanced testing)

- **Python Version:**  
  Python 3.9 or above

- **Privileges:**  
  Administrator privileges are required for packet capture

- **Internet Connection:**  
  Required to install dependencies and generate live network traffic
